# NetSuite Deposit Location Updater

This project contains a NetSuite SuiteScript 1.0 scheduled script used to remove location values from Deposit transactions and related line items.

## Features
- Loads transactions using a saved search
- Updates Deposit records
- Clears header-level location field
- Clears line-level locations for:
  - Other lines
  - Cashback lines

## Technology
- NetSuite SuiteScript 1.0
- Scheduled Script

## Notes
This project is intended for learning and portfolio purposes.
